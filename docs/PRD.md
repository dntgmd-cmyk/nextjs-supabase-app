# 모임 이벤트 관리 웹 MVP PRD

## 핵심 정보

**목적**: 수영, 헬스, 친구 모임 등 소/중규모 모임(2~30명) 주최자가 공지, 참여자 관리, 카풀, 정산을 한 곳에서 처리할 수 없어 발생하는 운영 비효율 해소

**사용자**: 정기적으로 소/중규모 모임을 운영하는 주최자, 그리고 해당 모임에 참여하는 일반 참여자

---

## 사용자 여정

```
1. 랜딩/홈 페이지 (비로그인)
   ↓ 로그인 버튼 클릭

2. 로그인 페이지
   ↓ 인증 성공

3. 내 모임 목록 페이지 (메인)
   ├── [주최자] → 모임 생성 페이지
   │              ↓ 저장 완료
   │              → 모임 상세 페이지
   │
   └── [모임 카드 클릭] → 모임 상세 페이지
                          ├── 개요 탭
                          ├── 참여자 탭
                          │    ├── [주최자] 승인/거절 처리 → 이메일 발송
                          │    └── [참여자] 참여 신청 → 이메일 발송(주최자)
                          ├── 공지 탭
                          │    ├── [주최자] 공지 작성 → 이메일 발송(승인된 참여자)
                          │    └── [참여자] 공지 목록 조회
                          ├── 카풀 탭
                          │    ├── [참여자] 운전자 등록 (좌석 수 입력)
                          │    ├── [참여자] 탑승 신청
                          │    └── [운전자] 탑승 확정/거절 → 이메일 발송
                          └── 정산 탭
                               ├── [주최자] 정산 생성 → 비용 항목 입력 → 이메일 발송
                               └── [참여자] 납부 여부 체크
```

## 기능 명세

### 1. MVP 핵심 기능

| ID       | 기능명      | 설명                                                        | MVP 필수 이유             | 관련 페이지                                                               |
| -------- | ----------- | ----------------------------------------------------------- | ------------------------- | ------------------------------------------------------------------------- |
| **F001** | 모임 CRUD   | 모임 생성/수정/삭제, 상태 관리(draft/open/closed/cancelled) | 서비스의 핵심 엔티티      | 내 모임 목록 페이지, 모임 생성 페이지, 모임 수정 페이지, 모임 상세 페이지 |
| **F002** | 참여자 관리 | 참여 신청, 주최자 승인/거절, 참여자 목록 조회               | 모임 운영의 핵심 흐름     | 모임 상세 페이지(참여자 탭), 참여자 관리 페이지                           |
| **F003** | 공지 관리   | 공지 작성, 조회, 이메일 일괄 발송                           | 모임 내 커뮤니케이션 핵심 | 모임 상세 페이지(공지 탭), 공지 목록/작성 페이지                          |
| **F004** | 카풀 매칭   | 운전자 등록(좌석 수), 탑승 신청, 운전자 확정/거절           | 모임 이동 조율 핵심 기능  | 모임 상세 페이지(카풀 탭), 카풀 목록/등록 페이지                          |
| **F005** | 정산        | 비용 항목 입력, 1/N 자동 계산, 납부 여부 체크               | 모임 후 비용 처리 핵심    | 모임 상세 페이지(정산 탭), 정산 상세 페이지                               |

### 2. MVP 필수 지원 기능

| ID       | 기능명             | 설명                                               | MVP 필수 이유                      | 관련 페이지                    |
| -------- | ------------------ | -------------------------------------------------- | ---------------------------------- | ------------------------------ |
| **F010** | 기본 인증          | 이메일 회원가입/로그인/로그아웃, 구글 OAuth 로그인 | 서비스 이용을 위한 최소 인증       | 로그인 페이지, 회원가입 페이지 |
| **F011** | 이메일 알림        | Resend를 통한 트리거별 이메일 자동 발송            | 모임 운영의 핵심 커뮤니케이션 수단 | (내부 서비스 - 별도 UI 없음)   |
| **F012** | 사용자 기본 프로필 | 이름, 이메일 (회원가입 시 입력)                    | 참여자 식별 및 이메일 발송에 필수  | 회원가입 페이지                |

### 3. MVP 이후 기능 (제외)

- 실제 결제/송금 API 연동 (토스, 카카오페이)
- 지도 연동 카풀 (카카오맵 등)
- 카카오톡 알림
- 모임 공개 탐색/검색 기능
- 프로필 상세 관리 (아바타, 자기소개)
- 소셜 기능 (팔로우, 좋아요)

## 메뉴 구조

```
내 모임 목록 페이지 (로그인 후 기본 화면)
├── 주최한 모임 탭 — F001 (내가 만든 모임 목록)
├── 참여한 모임 탭 — F001, F002 (내가 참여 신청/승인된 모임 목록)
└── 모임 만들기 버튼 — F001 (모임 생성 페이지로 이동)

모임 상세 페이지 (모임 카드 클릭 시 진입)
├── 개요 탭 — F001 (모임 기본 정보, 상태 표시)
│    └── [주최자만] 수정/삭제/상태 변경, 참여 신청 버튼 — F001, F002
├── 참여자 탭 — F002 (참여자 목록, 승인 대기 목록)
│    └── [주최자만] 승인/거절 버튼 — F002
├── 공지 탭 — F003 (공지 목록)
│    └── [주최자만] 공지 작성 버튼 — F003
├── 카풀 탭 — F004 (카풀 목록)
│    ├── [참여자] 운전자 등록/탑승 신청 버튼 — F004
│    └── [운전자] 탑승 확정/거절 버튼 — F004
└── 정산 탭 — F005 (정산 목록)
     ├── [주최자만] 정산 생성 버튼 — F005
     └── 정산 상세 링크 — F005

헤더 공통 메뉴
├── 로고/홈 — 내 모임 목록 페이지로 이동
├── (로그인 전) 로그인 — F010
├── (로그인 전) 회원가입 — F010
└── (로그인 후) 로그아웃 — F010
```

## 페이지별 상세 기능

### 로그인 페이지

> **구현 기능:** `F010` | **접근:** 비로그인 사용자, 보호 페이지 접근 시 자동 리디렉션

| 항목            | 내용                                                                                                                             |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 서비스 진입을 위한 인증 전용 페이지                                                                                              |
| **진입 경로**   | 헤더 로그인 버튼 클릭, 보호 페이지 접근 시 자동 리디렉션                                                                         |
| **사용자 행동** | 이메일/비밀번호 입력 후 로그인, 구글 계정으로 소셜 로그인                                                                        |
| **주요 기능**   | - 이메일 + 비밀번호 로그인 폼<br>- 구글 OAuth 로그인 버튼<br>- 비밀번호 찾기 링크<br>- 회원가입 페이지 링크<br>- **로그인** 버튼 |
| **다음 이동**   | 성공 → 내 모임 목록 페이지, 실패 → 에러 메시지 표시                                                                              |

---

### 회원가입 페이지

> **구현 기능:** `F010`, `F012` | **접근:** 비로그인 사용자

| 항목            | 내용                                                                                                                    |
| --------------- | ----------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 신규 사용자 계정 생성 전용 페이지                                                                                       |
| **진입 경로**   | 로그인 페이지의 회원가입 링크 클릭                                                                                      |
| **사용자 행동** | 이름, 이메일, 비밀번호를 입력하여 계정 생성                                                                             |
| **주요 기능**   | - 이름 입력 필드<br>- 이메일 입력 필드<br>- 비밀번호 입력 필드 (확인 포함)<br>- 이메일 중복 검사<br>- **회원가입** 버튼 |
| **다음 이동**   | 성공 → 이메일 인증 안내 화면, 실패 → 에러 메시지 표시                                                                   |

---

### 내 모임 목록 페이지

> **구현 기능:** `F001`, `F002` | **인증:** 로그인 필수

| 항목            | 내용                                                                                                                                                                                                                                                   |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **역할**        | 로그인 후 기본 랜딩 페이지. 내가 주최하거나 참여한 모임을 한눈에 파악                                                                                                                                                                                  |
| **진입 경로**   | 로그인 성공 후 자동 이동, 헤더 로고 클릭                                                                                                                                                                                                               |
| **사용자 행동** | 주최한 모임 / 참여한 모임 탭 전환, 모임 카드 클릭으로 상세 진입, 새 모임 생성 버튼 클릭                                                                                                                                                                |
| **주요 기능**   | - 주최한 모임 탭 (내가 만든 모임 카드 목록: 제목, 날짜, 장소, 상태, 참여자 수)<br>- 참여한 모임 탭 (내 참여 상태 배지 포함: 대기중/승인/거절)<br>- 모임 상태별 필터 (전체/모집중/마감/취소)<br>- 빈 상태 안내 (모임 없을 때)<br>- **모임 만들기** 버튼 |
| **다음 이동**   | 모임 카드 클릭 → 모임 상세 페이지, 모임 만들기 → 모임 생성 페이지                                                                                                                                                                                      |

### 모임 생성 페이지

> **구현 기능:** `F001` | **인증:** 로그인 필수

| 항목            | 내용                                                                                                                                                                                                                  |
| --------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 새 모임 정보를 입력하여 모임을 개설하는 페이지                                                                                                                                                                        |
| **진입 경로**   | 내 모임 목록 페이지의 모임 만들기 버튼 클릭                                                                                                                                                                           |
| **사용자 행동** | 모임 제목, 설명, 날짜, 장소, 최대 인원, 참여 방식을 입력하고 저장                                                                                                                                                     |
| **주요 기능**   | - 제목 입력 필드<br>- 설명 입력 필드 (텍스트에어리어)<br>- 날짜/시간 입력 필드<br>- 장소 입력 필드<br>- 최대 인원 입력 (2~30명)<br>- 참여 방식 선택 (자동 승인 / 주최자 승인)<br>- 초안으로 저장 / **모집 시작** 버튼 |
| **다음 이동**   | 저장 성공 → 모임 상세 페이지(개요 탭), 취소 → 내 모임 목록 페이지                                                                                                                                                     |

### 모임 수정 페이지

> **구현 기능:** `F001` | **인증:** 주최자만 접근 가능

| 항목            | 내용                                                                                                                                                                       |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 기존 모임 정보를 수정하는 페이지                                                                                                                                           |
| **진입 경로**   | 모임 상세 페이지(개요 탭)의 수정 버튼 클릭                                                                                                                                 |
| **사용자 행동** | 기존 모임 정보를 수정하고 저장 또는 모임 삭제/취소 처리                                                                                                                    |
| **주요 기능**   | - 모임 생성 페이지와 동일한 입력 폼 (기존 값 프리필)<br>- 모임 상태 변경 (draft → open → closed / cancelled)<br>- 모임 삭제 버튼 (확인 다이얼로그 포함)<br>- **저장** 버튼 |
| **다음 이동**   | 저장 성공 → 모임 상세 페이지(개요 탭), 삭제 → 내 모임 목록 페이지                                                                                                          |

### 모임 상세 페이지 - 개요 탭

> **구현 기능:** `F001`, `F002` | **인증:** 로그인 필수

| 항목            | 내용                                                                                                                                                                                                                                                                              |
| --------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 모임의 핵심 정보를 보여주는 메인 탭. 참여 신청의 진입점                                                                                                                                                                                                                           |
| **진입 경로**   | 내 모임 목록 페이지에서 모임 카드 클릭                                                                                                                                                                                                                                            |
| **사용자 행동** | 모임 정보 확인, 참여 신청(참여자) 또는 모임 수정(주최자), 탭 전환                                                                                                                                                                                                                 |
| **주요 기능**   | - 모임 기본 정보 표시 (제목, 설명, 날짜, 장소, 최대인원, 현재 승인 인원, 상태 배지)<br>- [참여자] 참여 신청 버튼 (미신청 시) / 신청 취소 버튼 (대기중 시) / 참여 상태 배지 (승인 시)<br>- [주최자] 수정 버튼, 모임 상태 변경 버튼<br>- 탭 내비게이션 (개요/참여자/공지/카풀/정산) |
| **다음 이동**   | 수정 클릭 → 모임 수정 페이지, 탭 클릭 → 해당 탭                                                                                                                                                                                                                                   |

| **진입 경로**   | 모임 상세 페이지의 참여자 탭 클릭                                                                                                                                                                                                               |
| **사용자 행동** | [주최자] 대기 중인 신청자 승인 또는 거절, 승인된 참여자 목록 확인 / [참여자] 참여자 목록 조회                                                                                                                                                   |
| **주요 기능**   | - 승인 대기 목록 섹션 (이름, 이메일, 신청일시)<br>- [주최자] 개별 승인/거절 버튼 (클릭 시 이메일 자동 발송)<br>- 승인된 참여자 목록 (이름, 이메일, 승인일시)<br>- 참여자 수 현황 표시 (승인 N명 / 최대 N명)<br>- [주최자] 참여자 강제 퇴장 버튼 |
| **다음 이동**   | 승인/거절 처리 → 현재 페이지 갱신 + 이메일 발송(F011)                                                                                                                                                                                           |

### 카풀 목록/등록 페이지

> **구현 기능:** `F004`, `F011` | **인증:** 승인된 참여자 및 주최자

| 항목            | 내용                                                                                                                                                                                                                                                                                     |
| --------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 카풀 운전자 등록과 탑승 신청을 처리하는 페이지                                                                                                                                                                                                                                           |
| **진입 경로**   | 모임 상세 페이지의 카풀 탭 클릭                                                                                                                                                                                                                                                          |
| **사용자 행동** | 운전자로 등록하거나 다른 운전자의 카풀에 탑승 신청. 운전자는 탑승 신청자 확정/거절                                                                                                                                                                                                       |
| **주요 기능**   | - 카풀 목록 (운전자명, 출발지, 가용 좌석, 현재 탑승 인원, 상태)<br>- 운전자 등록 폼 (출발지, 총 좌석 수, 메모)<br>- 탑승 신청 버튼 (좌석 있는 카풀에만 노출)<br>- [운전자] 탑승 신청자 목록 + 확정/거절 버튼 (확정 시 이메일 발송)<br>- 내 카풀 등록 취소 버튼<br>- **운전자 등록** 버튼 |
| **다음 이동**   | 탑승 확정 → 현재 페이지 갱신 + 이메일 발송(F011)                                                                                                                                                                                                                                         |

### 정산 상세 페이지

> **구현 기능:** `F005`, `F011` | **인증:** 승인된 참여자 및 주최자

| 항목            | 내용                                                                                                                                                                                                                                                                                                                                                                                                             |
| --------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 모임 비용을 항목별로 입력하고 1/N 계산 후 납부 현황을 관리하는 페이지                                                                                                                                                                                                                                                                                                                                            |
| **진입 경로**   | 모임 상세 페이지의 정산 탭에서 정산 상세 링크 클릭, 또는 주최자가 정산 생성 후 자동 이동                                                                                                                                                                                                                                                                                                                         |
| **사용자 행동** | [주최자] 비용 항목 입력, 정산 확정 및 이메일 발송 / [참여자] 내 납부 금액 확인 및 납부 여부 체크                                                                                                                                                                                                                                                                                                                 |
| **주요 기능**   | - 정산 제목 및 상태 표시 (open/closed)<br>- [주최자] 비용 항목 추가 폼 (항목명, 금액)<br>- 비용 항목 목록 및 합계 금액<br>- 1/N 자동 계산 결과 (승인된 참여자 수 기준)<br>- 참여자별 납부 현황 목록 (이름, 납부 금액, 납부 여부)<br>- [참여자] 내 납부 완료 체크 버튼<br>- [주최자] 참여자별 납부 완료 수동 처리 버튼<br>- [주최자] 정산 요청 이메일 발송 버튼 (승인된 참여자 전원)<br>- [주최자] 정산 마감 버튼 |
| **다음 이동**   | 정산 요청 이메일 → 이메일 발송(F011) + 현재 페이지 갱신, 마감 처리 → 현재 페이지 갱신                                                                                                                                                                                                                                                                                                                            |

## 이메일 알림 명세 (F011)

| 트리거         | 발송 시점                            | 수신자             | 발송 도구 |
| -------------- | ------------------------------------ | ------------------ | --------- |
| 참여 신청 접수 | 참여자가 신청 완료 시                | 주최자             | Resend    |
| 참여 승인      | 주최자가 승인 처리 시                | 신청자             | Resend    |
| 참여 거절      | 주최자가 거절 처리 시                | 신청자             | Resend    |
| 공지 등록      | 공지 작성 + 이메일 발송 옵션 선택 시 | 승인된 참여자 전원 | Resend    |
| 정산 요청      | 주최자가 정산 요청 발송 버튼 클릭 시 | 승인된 참여자 전원 | Resend    |
| 카풀 탑승 확정 | 운전자가 탑승 확정 처리 시           | 해당 탑승 신청자   | Resend    |

## 기술 스택

### 프론트엔드 프레임워크

- **Next.js 15** (App Router) - React 풀스택 프레임워크
- **TypeScript 5.6+** - 타입 안전성 보장
- **React 19** - UI 라이브러리 (최신 동시성 기능)

### 스타일링 & UI

- **TailwindCSS v4** (설정파일 없는 새로운 엔진) - 유틸리티 CSS 프레임워크
- **shadcn/ui** - 고품질 React 컴포넌트 라이브러리
- **Lucide React** - 아이콘 라이브러리

### 폼 & 검증

- **React Hook Form 7.x** - 폼 상태 관리
- **Zod** - 스키마 검증 라이브러리

### 백엔드 & 데이터베이스

- **Supabase** - BaaS (인증, PostgreSQL, Row Level Security)
- **PostgreSQL** - 관계형 데이터베이스 (Supabase 포함)

### 이메일 발송

- **Resend** - 트랜잭션 이메일 발송 서비스
- **React Email** - 이메일 템플릿 컴포넌트 (선택)

### 배포 & 호스팅

- **Vercel** - Next.js 15 최적화 배포 플랫폼

### 패키지 관리

- **npm** - 의존성 관리
